# 💡 Light Switch

A simple and interactive Light Switch project that allows users to toggle a virtual light ON and OFF with an attractive user interface.

![CSS](https://img.shields.io/badge/CSS-3-blue)

![License](https://img.shields.io/badge/License-MIT-green)

## 📌 Overview

Light Switch is a beginner-friendly web development project built using HTML, CSS, and JavaScript. It demonstrates DOM manipulation and event handling by allowing users to switch a light bulb ON and OFF interactively.

---

## ✨ Features

- 💡 Turn light ON/OFF
- 🎨 Responsive UI Design
- ⚡ Fast and Lightweight
- 📱 Mobile Friendly
- 🖱️ Interactive Button Controls

---

## 🎯 How It Works

1. User clicks the switch button.
2. JavaScript detects the click event.
3. The bulb image changes:
   - OFF ➜ ON
   - ON ➜ OFF
4. UI updates instantly.
